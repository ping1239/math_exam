## 선택된 접근법: 모던 에듀테크 (Modern EduTech)

### Design Movement
Korean Academic Precision — 한국 교육 문화의 체계성과 현대 디지털 학습 환경을 결합

### Core Principles
1. **집중력 우선**: 시험 환경에서 불필요한 시각적 노이즈 제거
2. **명확한 계층 구조**: 문제 번호, 배점, 문제 내용의 시각적 위계
3. **즉각적 피드백**: 채점 결과를 명확하고 직관적으로 표시
4. **수학적 정밀함**: 수식 표현의 정확성과 가독성

### Color Philosophy
- **Primary**: 딥 네이비 (#1a2744) — 신뢰, 학문, 집중
- **Accent**: 앰버 골드 (#f59e0b) — 강조, 배점, 중요 정보
- **Success**: 에메랄드 (#10b981) — 정답
- **Error**: 로즈 레드 (#ef4444) — 오답
- **Background**: 크림 화이트 (#fafaf8) — 시험지 느낌

### Layout Paradigm
- 좌측 고정 사이드바: 문제 번호 네비게이션 + 진행 상황
- 우측 메인 영역: 문제 내용 + 답안 선택
- 상단 헤더: 시험 정보 + 타이머 + 채점 버튼

### Signature Elements
1. 문제 카드: 좌측 컬러 보더 + 배점 배지
2. 선택지 호버 효과: 슬라이드 인 하이라이트
3. 채점 결과 애니메이션: 정답/오답 플립 카드

### Typography System
- 헤더: Noto Serif KR (700) — 권위감
- 본문: Noto Sans KR (400/500) — 가독성
- 수식: 인라인 수학 표현 (상위/하위 첨자 처리)
- 배점: Tabular nums, 모노스페이스 느낌

### Brand Essence
"정확한 학습, 명확한 피드백" — 중학생을 위한 수학 시험 플랫폼

### Brand Voice
- 헤드라인: "실력을 확인하세요"
- CTA: "채점하기", "풀이 보기"
- 마이크로카피: "문제를 선택하세요", "정답입니다!"
